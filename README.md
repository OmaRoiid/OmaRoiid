### About

Fullstack Engineer shipping product-focused web apps. My depth is on the frontend — React, TypeScript, Angular — and I build backends with NestJS and Node to own features end to end. The gap between a feature that *works* and one that actually *feels right* is usually in the details, and that's where I like to spend my time.
